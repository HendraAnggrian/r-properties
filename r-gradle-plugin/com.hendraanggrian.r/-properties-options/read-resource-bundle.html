readResourceBundle

fun readResourceBundle(readResourceBundle: Boolean): Unit

Groovy-friendly alias of readResourceBundle.

var readResourceBundle: Boolean

When set to true, will convert properties files that are resource bundle. Resource bundle files usually have suffix like _en, _id.

Default value is true, resembles to what works in JavaFX.
